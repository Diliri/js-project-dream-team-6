## Description

**«Хатинка лапок»** — це затишний та сучасний вебдодаток, створений для допомоги
безпритульним тваринам знайти свій новий дім та люблячу родину. Проєкт
покликаний спростити процес адаптації тварин, ознайомити користувачів із
діяльністю притулку та надати зручний інтерфейс для зв'язку з волонтерами.

### Особливості проєкту:

🛠 Реалізовано на **Vanilla JavaScript** з використанням сучасного інструменту
збірки **Vite**; 🛠 Повністю адаптивний та чуйний дизайн (Responsive Design) для
комфортного користування на смартфонах 📱, планшетах 📟 та десктопах 💻; 🛠
Використання анімацій та інтерактивних елементів для покращення користувацького
досвіду (UX).

Ми — студенти курсу FullStack Developer у [GoIT](https://goit.ua), і цей проєкт
є результатом нашої командної роботи наприкінці вивчення блоку JavaScript.

---

## Основний функціонал (Features)

🐾 **Каталог хвостиків:** зручний перегляд та фільтрація тварин притулку
(Котики, Собачки, Всі) з можливістю завантаження додаткових карток ("Показати
більше"). 🐾 **Історії успіху:** інтерактивний слайдер із реальними відгуками та
історіями людей, які вже знайшли свого друга. 🐾 **Інтерактивний FAQ:** секція
поширених запитань із плавним розкриттям відповідей (Accordion). 🐾 **Форма
зв'язку:** модальне вікно для залишення заявки на знайомство з тваринкою, що
містить валідацію введених користувачем даних. 🐾 **Скрол на початок сторінки:**
кнопка у вигляді лапки, яка повертає користувачів до хедеру сторінки. 🐾
**Світла та нічна тема:** кнопка, що береже ваш зір день та вночі.

## Технологічний стек (Tech Stack)

- HTML5 / CSS3
- Vanilla JavaScript (ES6+)
- Vite (збірник проєкту)
- Swiper.js (для реалізації слайдерів)
- інші бібліотеки, що використовували, Axios, Lodash

## Як запустити проєкт локально?

1. Склонуйте репозиторій:
   ```bash
   git clone [https://github.com/diliri/js-project-dream-team-6.git](https://github.com/diliri/js-project-dream-team-6.git)
   ```
2. Перейдіть у папку проєкту:
   ```bash
   cd js-project-dream-team-6
   ```
3. Встановіть залежності:
   ```bash
   npm install
   ```
4. Запустіть сервер для розробки:
   ```bash
   npm run dev
   ```
   💡 Живий сайт:
   [Переглянути Хатинку лапок](https://diliri.github.io/js-project-dream-team-6/)
